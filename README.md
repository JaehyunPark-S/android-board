# 4. 핵심 기능
- DB연동: SQLite3를 프로젝트에 연동한다.
- 로그인: 아이디 혹은 비밀번호 틀렸을 경우 예외처리
- 회원가입: 아이디 중복확인, 비밀번호와 비밀번호 확인의 불일치 예외처리
- 게시글 작성: 공백에 관한 예외처리
- 내 위치 찾기: 구글맵을 사용한 나의 위치를 알려주는 기능
